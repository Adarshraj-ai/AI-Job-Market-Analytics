## 📌 Overview

The **AI Job Market Analytics Dashboard** explores hiring patterns across the global AI job market using a dataset of 10,000 job postings. It answers questions like: which countries are hiring the most AI talent, what salaries look like across experience levels, which companies are hiring aggressively, and which skills actually get you hired.

Built end-to-end in Power BI — from raw Excel data to a polished, interactive 3-dashboard report — this project demonstrates data cleaning, data modeling, DAX measures, and dashboard design.

## 🚀 Getting Started

### Prerequisites
- [Microsoft Power BI Desktop](https://powerbi.microsoft.com/desktop/) (free)

### Installation

```bash
# 1. Clone the repository
git clone https://github.com/Adarshraj-ai/AI-Job-Market-Analytics.git

# 2. Navigate into the project
cd AI-Job-Market-Analytics

# 3. Open the .pbix file in Power BI Desktop
# 4. Refresh the dataset if prompted
# 5. Explore the dashboards using the interactive filters
```
